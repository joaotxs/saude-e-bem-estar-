# saude-e-bem-estar-
Sementes do Bem é uma página cyberpunk que une saúde e biotecnologia. Com visual futurista em tons neon de azul, ciano e roxo, o site ensina sobre bem-estar integrando elementos do campo à inovação digital. Possui cards expansíveis, tabela de hábitos, área de comentários e um painel de acessibilidade com leitura por voz e modo claro/escuro.
